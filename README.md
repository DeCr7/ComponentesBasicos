## 📱 Descripción de la aplicación

La aplicación desarrollada permite al usuario ingresar su nombre mediante un campo de texto y, al presionar un botón, mostrar un saludo personalizado en pantalla. Además, incluye una imagen que se despliega únicamente cuando se genera el saludo, reforzando visualmente la interacción. También se implementó un botón para alternar entre modo claro y oscuro, aprovechando las capacidades de Material Design 3, lo que mejora la experiencia visual del usuario y adapta la interfaz según sus preferencias.

## 🔧 Cambios realizados tras la validación

Luego de presentar la aplicación, se realizaron mejoras enfocadas en la usabilidad y claridad de la interfaz. Se ajustó el espaciado entre componentes para evitar saturación visual, se centraron los elementos para una mejor organización y se aumentó el tamaño del texto del saludo para hacerlo más legible. También se añadió validación en el campo de entrada para evitar que el usuario envíe datos vacíos, y se deshabilitó el botón de acción hasta que se ingrese información válida. Finalmente, se optimizó la interacción limpiando el campo de texto tras generar el saludo y ocultando la imagen hasta que exista una respuesta, logrando una experiencia más intuitiva y fluida.
